# AI-Powered Recommendation & Candidate Matching Engine 🚀
*An enterprise-grade AI system designed for the Ezitech Internship Portal to automate candidate profile evaluation, semantic matching, and mentor allocation.*

---

## 📌 Project Overview
Ezitech receives numerous internship applications with diverse technical backgrounds, resumes, and portfolios. This system replaces manual, slow review processes with an advanced AI pipeline that performs:
* **Resume Parsing & Skill Extraction**
* **Embedding-Based Semantic Search (via ChromaDB & Sentence Transformers)**
* **GitHub & Portfolio Analysis**
* **Confidence Scoring & AI-Driven Reasoning**
* **Personalized Learning Roadmap Generation**

---

## 🛠️ Tech Stack
* **Backend Framework:** FastAPI (Python)
* **Database & ORM:** PostgreSQL, SQLAlchemy, ChromaDB (Vector Store)
* **AI & NLP Models:** Sentence Transformers (`all-MiniLM-L6-v2`)
* **External Integrations:** GitHub REST API for repository metrics
